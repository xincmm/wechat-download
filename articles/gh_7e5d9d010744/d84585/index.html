深夜王炸！

刚刚Sam Altman亲自放出了一个“王炸”级别的roadmap更新， 废话不多说，赶紧上车，第一时间带你解读：

一句话总结：OpenAI要搞大事！简化产品线，推出更强更统一的GPT-5，而且免费用户竟然也能用上？！

这次更新的核心关键词就一个字：统一！我们先来看看Sam Altman的原话，我给大家划重点：

“我们想要改进路线图的分享，并大大简化我们的产品供应。”

“我们希望AI对你来说‘开箱即用’；我们意识到我们的模型和产品供应变得多么复杂。”

“我们和你们一样讨厌模型选择器，并希望回归神奇的统一智能。”

Sam自己也承认现在的产品线太复杂了，又是GPT系列又是o系列，用户选择困难症都犯了！他们也受够了那个让人头疼的模型选择器，目标是回归到 “统一智能”，让AI用起来更简单、更“魔法”！

接下来，才是真正的重磅炸弹！ Sam Altman 亲自剧透了GPT-4.5 和 GPT-5 的未来规划：

1. GPT-4.5（代号 Orion）：最后的“非链式思考”模型

“我们将接下来发布 GPT-4.5，我们在内部称之为 Orion 的模型，作为我们最后一个非链式思考模型。”

GPT-4.5 将是 GPT-4 的升级版，性能肯定会更强，但它依然属于我们现在熟悉的这种“传统”模型。重点是 “最后一个非链式思考模型”！继续往下看！

2. GPT-5：统一 o 系列和 GPT 系列的“终极形态”

“在那之后，我们的首要目标是统一 o 系列模型和 GPT 系列模型，创建可以使用我们所有工具的系统，知道何时长时间思考或不思考，并且通常适用于非常广泛的任务。”

“在 ChatGPT 和我们的 API 中，我们将发布 GPT-5 作为一个集成了我们大量技术的系统，包括 o3。我们将不再单独发布 o3 作为独立模型。”

GPT-5 将不再仅仅是一个模型，而是一个 “系统”！它将整合 OpenAI 所有的技术。能够 “使用所有工具”，并且能够 “知道何时长时间思考或不思考”。这不就是我们一直期待的 “更智能、更像人” 的AI吗？！

OpenAI 想要打造一个 “All-in-One” 的 AI 系统，GPT-5 就是这个 “终极形态”的起点

3. 免费用户狂喜！GPT-5 免费用！

“ChatGPT 的免费层级将获得无限次的 GPT-5 聊天访问权限，采用标准智能设置，但会受到滥用阈值的限制。”

🤯 看到这里我直接惊呼 “卧槽”！ 免费用户竟然也能用上 GPT-5 了？！ 虽然是 “标准智能设置”，而且会有 “滥用阈值”，但这已经足够震撼了好吗！这意味着，即使不花钱，我们也能体验到 OpenAI 最新的、最强大的 AI 技术！感谢deepseek？

4. 付费订阅更上一层楼：更高智能 + 更多功能！

“Plus 订阅用户将能够以更高的智能水平运行 GPT-5，Pro 订阅用户将能够以更高的智能水平运行 GPT-5。这些模型将整合语音、画布、搜索、深度研究等等。”

当然，OpenAI 也要恰饭的！Plus 和 Pro 订阅用户将获得 更高智能水平 的 GPT-5，并且还将解锁更多酷炫的功能，例如 语音、画布、搜索、深度研究 等等。这无疑会进一步提升付费用户的体验，让他们享受到更强大的 AI 能力

5. 发布时间线：Weeks / Months

最后，大家最关心的 ETA （预计发布时间）来了！Chubby 问 Sam Altman：“GPT 4.5 / GPT 5 的 ETA 是多久？周？月？” Sam Altman 的回答是： “weeks / months”！

这意味着什么？ GPT-4.5 和 GPT-5 很有可能在几周到几个月内就会发布！

全文如下：

萨姆·奥特曼

OPENAI GPT-4.5 与 GPT-5 路线图更新：

我们希望在分享我们的规划蓝图方面做得更好，并在大幅简化我们的产品供应方面做得更好。

我们希望人工智能对您来说能够“开箱即用”；我们意识到我们的模型和产品已经变得多么复杂。

我们和你们一样讨厌模型选择器，并且希望回归到神奇的统一智能。

接下来，我们将发布 GPT-4.5，这个我们在内部称之为 Orion 的模型，这将是我们最后一个非链式思考模型。

在那之后，我们的首要目标是统一 o 系列模型和 GPT 系列模型，创建能够使用我们所有工具的系统，使其知道何时需要长时间思考，何时不需要，并且总体上能够广泛适用于各种任务。

在 ChatGPT 和我们的 API 中，我们将发布 GPT-5，作为一个整合了我们大量技术的系统，包括 o3。我们将不再单独发布 o3 作为独立模型。

ChatGPT 的免费层级将获得无限次的 GPT-5 聊天访问权限，采用标准智能设置（！！），当然会受到滥用阈值的限制。

Plus 订阅用户将能够以更高的智能水平运行 GPT-5，而 Pro 订阅用户将能够以更高的智能水平运行 GPT-5。这些模型将整合语音、画布、搜索、深度研究以及更多功能
