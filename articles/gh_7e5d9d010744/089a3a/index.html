刚刚OpenAI正式推出o3 mini：免费用户也可以用「感谢DeepSeek」

AI寒武纪
2025年02月01日 08:38 陕西


 

刚刚OpenAI 正式推出了 o3-mini 模型，主打 高性价比推理，旨在推动更广泛的 AI 应用，尤其是在 STEM 领域（科学、技术、工程、数学）！

划重点！o3-mini 到底有多强？

根据 OpenAI 官方博客 (我第一时间就去扒了原文！)，o3-mini 定位为他们“推理系列”中的最新成员，并且是目前 成本效益最高 的模型。今天在 ChatGPT 和 API 中同步上线，实力主要表现如下：

  • • 卓越的 STEM 能力： o3-mini 在科学、数学和编程方面表现出色，超越之前的 o1-mini 模型，同时还保持了低成本和低延迟的优势

  • • 强大的功能特性： o3-mini 支持开发者们呼声很高的功能，例如 函数调用 (Function Calling)、结构化输出 (Structured Outputs) 和 开发者消息 (Developer Messages)。这意味着 o3-mini 开箱即用，生产力爆棚！同时，它也像 o1-mini 一样支持流式传输 (Streaming)

  • • 灵活的推理模式： 开发者可以根据不同的使用场景，选择 低、中、高 三种推理努力程度。当你需要“硬核思考”解决复杂难题时，可以选择高推理努力；当你更注重速度和低延迟时，可以选择低推理努力

  • • 更快的速度，更低的延迟： 官方数据显示，o3-mini 的平均响应时间比 o1-mini 快了 24%，首个 token 的生成速度也快了 2.5 秒 (2500ms)。这意味着更流畅的交互体验，效率大大提升！

  • • 更强的性能表现： 在多项基准测试中，o3-mini 都表现出色。例如，在数学、博士级科学问题、代码竞赛和软件工程等领域，o3-mini 都展现出了超越 o1-mini 的能力。尤其是在 Codeforces 代码竞赛 和 SWE-bench 软件工程基准 上，o3-mini 成为了 OpenAI 目前表现最佳的模型！

用户如何体验 o3-mini？

  • • ChatGPT 用户： ChatGPT Plus、Team 和 Pro 用户今天就可以体验 o3-mini 了！企业版用户预计 2 月份也能用上。免费 ChatGPT 用户 也可以通过在消息编辑器中选择 “Reason” 功能或重新生成回复来尝试 o3-mini！这真的是首次向免费用户开放推理模型（感谢一下DeepSeek）

  • • API 开发者： o3-mini 已经集成到 Chat Completions API、Assistants API 和 Batch API 中，今天开始面向 API 使用级别 3-5 的开发者开放

速率限制提升！

对于 ChatGPT Plus 和 Team 用户，OpenAI 将 o3-mini 的每日消息速率限制从原来的 50 条提升到了 150 条！（再次感谢DeepSeek）

o3-mini vs o1-mini

o3-mini 将会逐步取代 o1-mini 成为模型选择器中的默认选项。虽然 o1-mini 仍然是一个通用的知识推理模型，但 o3-mini 明显更专注于技术领域，在精度和速度上都更有优势。对于需要更高智能版本的用户，付费用户还可以选择 o3-mini-high 版本，虽然响应时间稍长，但智能程度更高！Pro 用户则可以无限制访问 o3-mini 和 o3-mini-high 两个版本。

写在最后：

感谢DeepSeek，由于deepseek r1 的发布，sam altman 才考虑把推理模型提供给免费用户，虽然有提问次数限制，但至少可以体验了，也可以同时开启搜索和推理了
